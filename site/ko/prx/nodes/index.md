# 원격 노드

PRX는 원격 노드를 통한 분산 에이전트 실행을 지원합니다. 노드는 별도의 머신에서 실행되는 PRX 인스턴스로, 위임된 작업 실행을 위해 컨트롤러와 페어링할 수 있습니다.

## 개요

노드 시스템은 다음을 가능하게 합니다:

- **분산 실행** -- 원격 머신에서 에이전트 작업 실행
- **특수 환경** -- GPU 접근, 특정 도구 또는 네트워크 위치를 갖춘 노드
- **부하 분산** -- 여러 머신에 에이전트 워크로드 분산
- **헤드리스 운영** -- 로컬 사용자 인터페이스 없이 데몬으로 노드 실행

## 아키텍처

```
┌──────────────┐         ┌──────────────┐
│  Controller  │◄──────► │   Node A     │
│  (primary)   │         │  (GPU host)  │
│              │         └──────────────┘
│              │         ┌──────────────┐
│              │◄──────► │   Node B     │
│              │         │  (staging)   │
└──────────────┘         └──────────────┘
```

## 설정

```toml
[node]
mode = "controller"  # "controller" | "node"
node_id = "gpu-host-01"
advertise_address = "192.168.1.100:3121"

[node.discovery]
method = "static"  # "static" | "mdns"
peers = ["192.168.1.101:3121"]
```
