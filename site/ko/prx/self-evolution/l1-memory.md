# L1: 메모리 진화

레이어 1은 가장 빈번하고 위험이 가장 낮은 자기 진화 계층입니다. 에이전트의 메모리 시스템에서 작동하며 중복된 항목을 자동으로 압축하고 관련된 메모리를 토픽별로 클러스터링합니다.

## 개요

L1 진화는 매 세션 후 (또는 설정 가능한 일정에 따라) 실행되며 다음을 수행합니다:

- **압축** -- 여러 관련 메모리 항목을 간결한 요약으로 병합
- **토픽 클러스터링** -- 시맨틱 유사도로 메모리를 그룹화
- **관련성 점수** -- 접근 빈도에 따라 메모리 가중치 조정
- **정리** -- 오래되었거나 모순된 메모리 제거

## 작동 방식

1. 세션 종료 후 L1이 새로 저장된 메모리를 분석합니다
2. 임베딩 유사도를 사용하여 관련 항목의 클러스터를 식별합니다
3. 크기 임계값을 초과하는 클러스터를 요약으로 압축합니다
4. 리콜 빈도에 따라 메모리 관련성 점수를 업데이트합니다

## 설정

```toml
[self_evolution.l1]
enabled = true
schedule = "after_session"  # 또는 "hourly", "daily"
compaction_threshold = 10
cluster_similarity = 0.8
min_access_count = 2
```
