# Метки

Метки обеспечивают гибкий способ категоризации и фильтрации задач. Каждая метка имеет название, цвет и опциональное описание.

## Создание меток

### Через веб-интерфейс

1. Перейдите в настройки проекта или рабочего пространства.
2. Перейдите в **Labels**.
3. Нажмите **New Label**.
4. Введите название (например, "bug", "feature", "documentation").
5. Выберите цвет (hex-формат, например, `#ef4444` для красного).
6. Нажмите **Create**.

### Через API

```bash
curl -X POST http://localhost:8080/api/labels \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "bug",
    "color": "#ef4444",
    "description": "Something is not working"
  }'
```

### Через MCP

```json
{
  "method": "tools/call",
  "params": {
    "name": "labels.create",
    "arguments": {
      "name": "bug",
      "color": "#ef4444"
    }
  }
}
```

## Распространённые схемы меток

Вот несколько популярных организаций меток:

### По типу

| Метка | Цвет | Описание |
|-------|------|----------|
| `bug` | `#ef4444` (красный) | Что-то сломано |
| `feature` | `#3b82f6` (синий) | Запрос новой функции |
| `enhancement` | `#8b5cf6` (фиолетовый) | Улучшение существующей функции |
| `documentation` | `#06b6d4` (голубой) | Обновления документации |
| `refactor` | `#f59e0b` (янтарный) | Рефакторинг кода |

### По приоритету

| Метка | Цвет | Описание |
|-------|------|----------|
| `P0-critical` | `#dc2626` (красный) | Продакшен недоступен |
| `P1-high` | `#ea580c` (оранжевый) | Критическая функция сломана |
| `P2-medium` | `#eab308` (жёлтый) | Некритическая проблема |
| `P3-low` | `#22c55e` (зелёный) | Желательно иметь |

## Добавление меток к задачам

### Через веб-интерфейс

Откройте задачу и нажмите на поле **Labels** для добавления или удаления меток.

### Через API

```bash
# Добавить метку к задаче
curl -X POST http://localhost:8080/api/issues/<issue_id>/labels \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{"label_id": "<label_uuid>"}'
```

### Через MCP

| Инструмент | Параметры | Описание |
|----------|---------|----------|
| `work_items.add_label` | `work_item_id`, `label_id` | Добавить одну метку |
| `work_items.add_labels` | `work_item_id`, `label_ids` | Добавить несколько меток |
| `work_items.remove_label` | `work_item_id`, `label_id` | Удалить метку |
| `work_items.list_labels` | `work_item_id` | Список меток задачи |

## MCP-инструменты управления метками

| Инструмент | Параметры | Описание |
|----------|---------|----------|
| `labels.list` | — | Список всех меток рабочего пространства |
| `labels.list_by_project` | `project_id` | Список меток проекта |
| `labels.create` | `name`, `color` | Создать метку |
| `labels.update` | `label_id` | Обновить название, цвет или описание |
| `labels.delete` | `label_id` | Удалить метку |
