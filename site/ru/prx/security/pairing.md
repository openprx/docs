# Сопряжение устройств

PRX использует модель сопряжения устройств для аутентификации экземпляров агента и установления доверия между узлами. Сопряжение гарантирует, что только авторизованные устройства могут подключаться к агенту и управлять им.

## Обзор

Процесс сопряжения:

1. Генерация уникальной идентичности устройства (пара ключей Ed25519)
2. Обмен открытыми ключами между контроллером и агентом
3. Верификация идентичности через протокол вызов-ответ
4. Установление зашифрованного канала связи

## Поток сопряжения

```
Контроллер                   Агент
    │                           │
    │──── Запрос сопряжения ──►│
    │                           │
    │◄─── Вызов ────────────── │
    │                           │
    │──── Подписанный ответ ──►│
    │                           │
    │◄─── Подтверждение ─────  │
```

## Конфигурация

```toml
[security.pairing]
require_pairing = true
max_paired_devices = 5
challenge_timeout_secs = 30
```

## Управление сопряжёнными устройствами

```bash
prx pair list          # Список сопряжённых устройств
prx pair add           # Начать поток сопряжения
prx pair remove <id>   # Удалить сопряжённое устройство
prx pair revoke-all    # Отозвать все сопряжения
```
