# prx evolution

Просмотр и управление движком самоэволюции. PRX поддерживает три уровня автономной эволюции: L1 (память), L2 (промпты) и L3 (стратегии). Эта команда позволяет проверять статус эволюции, просматривать историю, обновлять конфигурацию и запускать циклы эволюции вручную.

## Использование

```bash
prx evolution <SUBCOMMAND> [OPTIONS]
```

## Подкоманды

### `prx evolution status`

Отображение текущего состояния движка эволюции.

```bash
prx evolution status [OPTIONS]
```

| Флаг | Сокр. | По умолчанию | Описание |
|------|-------|--------------|----------|
| `--json` | `-j` | `false` | Вывод в формате JSON |

**Пример вывода:**

```
 Evolution Engine Status
 ───────────────────────
 Engine:    running
 L1 Memory:    enabled   (last: 2h ago, next: in 4h)
 L2 Prompt:    enabled   (last: 1d ago, next: in 23h)
 L3 Strategy:  disabled
 Total cycles: 142
 Rollbacks:    3
```

### `prx evolution history`

Показать журнал истории эволюции.

```bash
prx evolution history [OPTIONS]
```

| Флаг | Сокр. | По умолчанию | Описание |
|------|-------|--------------|----------|
| `--limit` | `-n` | `20` | Количество записей для отображения |
| `--level` | `-l` | все | Фильтр по уровню: `l1`, `l2`, `l3` |
| `--json` | `-j` | `false` | Вывод в формате JSON |

```bash
# Показать последние 10 эволюций уровня L2
prx evolution history --limit 10 --level l2
```

**Пример вывода:**

```
 Time                Level  Action                          Status
 2026-03-21 08:00    L1     memory consolidation            success
 2026-03-20 20:00    L1     memory consolidation            success
 2026-03-20 09:00    L2     prompt refinement (system)      success
 2026-03-19 14:22    L2     prompt refinement (tool-use)    rolled back
```

### `prx evolution config`

Просмотр или обновление конфигурации эволюции.

```bash
prx evolution config [OPTIONS]
```

| Флаг | Сокр. | По умолчанию | Описание |
|------|-------|--------------|----------|
| `--set` | | | Установить значение конфигурации (например, `--set l1.enabled=true`) |
| `--json` | `-j` | `false` | Вывод в формате JSON |

```bash
# Просмотр текущей конфигурации
prx evolution config

# Включить эволюцию стратегий L3
prx evolution config --set l3.enabled=true

# Установить интервал L1 на 2 часа
prx evolution config --set l1.interval=7200
```

### `prx evolution trigger`

Ручной запуск цикла эволюции.

```bash
prx evolution trigger [OPTIONS]
```

| Флаг | Сокр. | По умолчанию | Описание |
|------|-------|--------------|----------|
| `--level` | `-l` | `l1` | Уровень эволюции для запуска: `l1`, `l2`, `l3` |
| `--dry-run` | | `false` | Предварительный просмотр эволюции без применения изменений |

```bash
# Запуск эволюции памяти L1
prx evolution trigger --level l1

# Предварительный просмотр эволюции промптов L2
prx evolution trigger --level l2 --dry-run
```

## Уровни эволюции

| Уровень | Цель | Описание |
|---------|------|----------|
| **L1** | Память | Консолидация, дедупликация и организация записей памяти |
| **L2** | Промпты | Совершенствование системных промптов и инструкций по использованию инструментов на основе паттернов взаимодействия |
| **L3** | Стратегии | Адаптация высокоуровневых поведенческих стратегий (требуется явное включение) |

Все изменения эволюции обратимы. Движок ведёт историю откатов и автоматически отменяет изменения, вызывающие ухудшение производительности.
