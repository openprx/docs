# prx onboard

Запуск мастера настройки для конфигурации PRX при первом использовании. Мастер проводит через выбор провайдера, настройку API-ключа, конфигурацию каналов и базовые предпочтения.

## Использование

```bash
prx onboard [OPTIONS]
```

## Параметры

| Флаг | Сокр. | По умолчанию | Описание |
|------|-------|--------------|----------|
| `--quick` | `-q` | `false` | Быстрый режим — минимум вопросов, разумные значения по умолчанию |
| `--provider` | `-P` | | Предварительный выбор провайдера (пропускает этап выбора провайдера) |
| `--config` | `-c` | `~/.config/prx/config.toml` | Путь для записи файла конфигурации |
| `--force` | `-f` | `false` | Перезаписать существующий файл конфигурации |
| `--non-interactive` | | `false` | Неинтерактивный режим (требуется `--provider` и переменные окружения для ключей) |

## Шаги мастера

Интерактивный мастер проводит вас через следующие шаги:

1. **Выбор провайдера** — выбор основного LLM-провайдера (Anthropic, OpenAI, Ollama и т.д.)
2. **Настройка API-ключа** — ввод и проверка вашего API-ключа
3. **Выбор модели** — выбор модели по умолчанию от выбранного провайдера
4. **Настройка каналов** (необязательно) — конфигурация одного или нескольких каналов обмена сообщениями
5. **Бэкенд памяти** — выбор хранилища для памяти разговоров (markdown, SQLite, PostgreSQL)
6. **Безопасность** — настройка кода сопряжения и предпочтений песочницы
7. **Обзор конфигурации** — предварительный просмотр сгенерированной конфигурации и подтверждение

## Примеры

```bash
# Полный интерактивный мастер
prx onboard

# Быстрая настройка с Anthropic
prx onboard --quick --provider anthropic

# Неинтерактивный режим (API-ключ из переменной окружения)
export ANTHROPIC_API_KEY="sk-ant-..."
prx onboard --non-interactive --provider anthropic

# Запись конфигурации по пользовательскому пути
prx onboard --config /etc/prx/config.toml

# Повторный запуск мастера (перезапись существующей конфигурации)
prx onboard --force
```

## Быстрый режим

Быстрый режим (`--quick`) пропускает необязательные шаги и использует разумные значения по умолчанию:

- Бэкенд памяти: SQLite
- Безопасность: песочница включена, сопряжение не требуется
- Каналы: нет (можно добавить позже с помощью `prx channel add`)
- Эволюция: отключена (можно включить позже в конфигурации)

Это самый быстрый способ получить рабочую конфигурацию:

```bash
prx onboard --quick --provider ollama
```

## После настройки

После завершения настройки вы можете:

```bash
# Проверить конфигурацию
prx doctor

# Начать чат
prx chat

# Добавить больше каналов
prx channel add

# Запустить полный демон
prx daemon
```
