# Система самоэволюции

PRX включает трёхуровневую систему самоэволюции, позволяющую агентам автономно улучшать своё поведение с течением времени. Система непрерывно анализирует производительность агента и применяет градуированные улучшения -- от оптимизации памяти до настройки промптов и стратегических изменений политик.

## Обзор

Самоэволюция организована в три уровня, каждый из которых работает на разном уровне абстракции:

| Уровень | Область | Частота | Риск |
|---------|---------|---------|------|
| [L1: Память](./l1-memory) | Компактизация памяти, кластеризация по темам | Каждая сессия | Низкий |
| [L2: Промпт](./l2-prompt) | Оптимизация системного промпта, A/B-тестирование | Ежедневно/еженедельно | Средний |
| [L3: Стратегия](./l3-strategy) | Политики инструментов, правила маршрутизации, управление | Еженедельно/ежемесячно | Высокий |

## Архитектура

```
┌───────────────────────────────────────┐
│         Движок самоэволюции           │
│                                        │
│  L3: Стратегия  ← Низкая частота      │
│    ├── Настройка политик инструментов │
│    ├── Оптимизация маршрутизации      │
│    └── Корректировка управления       │
│                                        │
│  L2: Промпт     ← Средняя частота     │
│    ├── Уточнение системного промпта   │
│    └── Фреймворк A/B-тестирования     │
│                                        │
│  L1: Память     ← Высокая частота     │
│    ├── Компактизация памяти           │
│    └── Кластеризация по темам         │
└───────────────────────────────────────┘
```

## Безопасность прежде всего

Каждое предложение по эволюции проходит через конвейер безопасности перед выполнением. Подробности о защите откатом и проверках корректности см. в разделе [Безопасность](./safety).

## Конфигурация

```toml
[self_evolution]
enabled = false  # только по явному согласию
auto_apply = false  # по умолчанию требуется ручное одобрение

[self_evolution.l1]
enabled = true
schedule = "after_session"

[self_evolution.l2]
enabled = false
schedule = "weekly"

[self_evolution.l3]
enabled = false
schedule = "monthly"
require_approval = true
```
