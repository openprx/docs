# L1: Эволюция памяти

Уровень 1 -- это наиболее частый и наименее рискованный уровень самоэволюции. Он работает с системой памяти агента, автоматически компактизируя избыточные записи и группируя связанные воспоминания по темам.

## Обзор

L1-эволюция запускается после каждой сессии (или по настраиваемому расписанию) и выполняет:

- **Компактизацию** -- объединение нескольких связанных записей памяти в краткие сводки
- **Кластеризацию по темам** -- группировку воспоминаний по семантическому сходству
- **Оценку релевантности** -- корректировку весов памяти на основе частоты обращений
- **Обрезку** -- удаление воспоминаний, ставших устаревшими или опровергнутыми

## Как это работает

1. После завершения сессии L1 анализирует вновь сохранённые воспоминания
2. Определяет кластеры связанных записей с помощью сходства эмбеддингов
3. Кластеры, превышающие порог размера, компактизируются в сводки
4. Баллы релевантности памяти обновляются на основе частоты вспоминания

## Конфигурация

```toml
[self_evolution.l1]
enabled = true
schedule = "after_session"  # или "hourly", "daily"
compaction_threshold = 10
cluster_similarity = 0.8
min_access_count = 2
```
