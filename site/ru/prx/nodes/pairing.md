# Сопряжение узлов

Прежде чем узел сможет получать задачи от контроллера, они должны быть сопряжены. Сопряжение устанавливает взаимное доверие через криптографическую верификацию идентичности.

## Процесс сопряжения

1. Запустите узел в режиме сопряжения: `prx node pair`
2. Узел отображает код сопряжения (6-значный PIN)
3. На контроллере инициируйте сопряжение: `prx pair add --address <node-ip>:3121`
4. Введите код сопряжения при запросе
5. Обе стороны обмениваются и верифицируют открытые ключи Ed25519

## Конфигурация

```toml
[node.pairing]
auto_accept = false
pairing_timeout_secs = 120
max_paired_controllers = 3
```

## Управление узлами

```bash
# На контроллере
prx node list              # Список сопряжённых узлов
prx node status <node-id>  # Проверка статуса узла
prx node unpair <node-id>  # Удаление сопряжения узла

# На узле
prx node pair              # Вход в режим сопряжения
prx node info              # Показать идентичность узла
```
