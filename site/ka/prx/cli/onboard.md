# prx onboard

გაუშვით დაყენების ოსტატი PRX-ის პირველადი გამოყენებისთვის კონფიგურაციისთვის. ოსტატი გაგატარებთ პროვაიდერის არჩევაში, API გასაღების დაყენებაში, არხების კონფიგურაციასა და ძირითად პარამეტრებში.

## გამოყენება

```bash
prx onboard [OPTIONS]
```

## ვარიანტები

| ფლაგი | მოკლე | ნაგულისხმევი | აღწერა |
|-------|-------|-------------|--------|
| `--quick` | `-q` | `false` | სწრაფი რეჟიმი -- მინიმალური მოთხოვნები, გონივრული ნაგულისხმევი მნიშვნელობები |
| `--provider` | `-P` | | პროვაიდერის წინასწარ არჩევა (პროვაიდერის არჩევის ნაბიჯის გამოტოვება) |
| `--config` | `-c` | `~/.config/prx/config.toml` | კონფიგურაციის ფაილის გამოტანის გზა |
| `--force` | `-f` | `false` | არსებული კონფიგურაციის ფაილის გადაწერა |
| `--non-interactive` | | `false` | არაინტერაქტიული რეჟიმი (მოითხოვს `--provider`-ს და გარემოს ცვლადებს გასაღებებისთვის) |

## ოსტატის ნაბიჯები

ინტერაქტიული ოსტატი გაგატარებთ შემდეგ ნაბიჯებზე:

1. **პროვაიდერის არჩევა** -- აირჩიეთ თქვენი ძირითადი LLM პროვაიდერი (Anthropic, OpenAI, Ollama და ა.შ.)
2. **API გასაღების კონფიგურაცია** -- შეიყვანეთ და დაადასტურეთ თქვენი API გასაღები
3. **მოდელის არჩევა** -- აირჩიეთ ნაგულისხმევი მოდელი არჩეული პროვაიდერიდან
4. **არხების დაყენება** (არასავალდებულო) -- დააკონფიგურირეთ ერთი ან მეტი შეტყობინებების არხი
5. **მეხსიერების ბექენდი** -- აირჩიეთ სად შეინახოთ საუბრის მეხსიერება (markdown, SQLite, PostgreSQL)
6. **უსაფრთხოება** -- დააყენეთ დაწყვილების კოდი და სენდბოქსის პარამეტრები
7. **კონფიგურაციის მიმოხილვა** -- გენერირებული კონფიგურაციის წინასწარი ნახვა და დადასტურება

## მაგალითები

```bash
# სრული ინტერაქტიული ოსტატი
prx onboard

# სწრაფი დაყენება Anthropic-ით
prx onboard --quick --provider anthropic

# არაინტერაქტიული (API გასაღები გარემოდან)
export ANTHROPIC_API_KEY="sk-ant-..."
prx onboard --non-interactive --provider anthropic

# კონფიგურაციის ჩაწერა მორგებულ გზაზე
prx onboard --config /etc/prx/config.toml

# ოსტატის ხელახლა გაშვება (არსებული კონფიგურაციის გადაწერა)
prx onboard --force
```

## სწრაფი რეჟიმი

სწრაფი რეჟიმი (`--quick`) გამოტოვებს არასავალდებულო ნაბიჯებს და იყენებს გონივრულ ნაგულისხმევ მნიშვნელობებს:

- მეხსიერების ბექენდი: SQLite
- უსაფრთხოება: სენდბოქსი ჩართული, დაწყვილება არ არის საჭირო
- არხები: არცერთი (მოგვიანებით დაამატეთ `prx channel add`-ით)
- ევოლუცია: გამორთული (მოგვიანებით ჩართეთ კონფიგურაციაში)

ეს არის მოქმედი კონფიგურაციის მიღების ყველაზე სწრაფი გზა:

```bash
prx onboard --quick --provider ollama
```

## დაყენების შემდეგ

ონბორდინგის დასრულების შემდეგ შეგიძლიათ:

```bash
# კონფიგურაციის შემოწმება
prx doctor

# ჩატის დაწყება
prx chat

# მეტი არხების დამატება
prx channel add

# სრული დემონის გაშვება
prx daemon
```
