# კვანძის დაწყვილება

სანამ კვანძი კონტროლერისგან ამოცანებს მიიღებს, ისინი დაწყვილებული უნდა იყოს. დაწყვილება ამყარებს ორმხრივ ნდობას კრიპტოგრაფიული იდენტობის ვერიფიკაციით.

## დაწყვილების პროცესი

1. კვანძის გაშვება დაწყვილების რეჟიმში: `prx node pair`
2. კვანძი აჩვენებს დაწყვილების კოდს (6-ციფრიანი PIN)
3. კონტროლერზე დაწყვილების ინიციირება: `prx pair add --address <node-ip>:3121`
4. მოთხოვნისას დაწყვილების კოდის შეყვანა
5. ორივე მხარე ცვლის და ამოწმებს Ed25519 საჯარო გასაღებებს

## კონფიგურაცია

```toml
[node.pairing]
auto_accept = false
pairing_timeout_secs = 120
max_paired_controllers = 3
```

## კვანძების მართვა

```bash
# კონტროლერზე
prx node list              # დაწყვილებული კვანძების ჩამონათვალი
prx node status <node-id>  # კვანძის სტატუსის შემოწმება
prx node unpair <node-id>  # კვანძის დაწყვილების მოხსნა

# კვანძზე
prx node pair              # დაწყვილების რეჟიმში შესვლა
prx node info              # კვანძის იდენტობის ჩვენება
```
