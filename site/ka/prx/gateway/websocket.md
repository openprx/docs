# WebSocket

PRX გეითვეი WebSocket ენდფოინთს უზრუნველყოფს რეალურ დროში, ორმხრივი კომუნიკაციისთვის აგენტის სესიებთან. ეს საშუალებას აძლევს სტრიმინგ პასუხებს, ინსტრუმენტების შესრულების ცოცხალ განახლებებსა და ინტერაქტიულ საუბრებს.

## კავშირი

დაუკავშირდით WebSocket ენდფოინთს:

```
ws://127.0.0.1:3120/ws/sessions/:id
```

## შეტყობინებების პროტოკოლი

შეტყობინებები JSON ობიექტების სახით იცვლება `type` ველით:

### კლიენტიდან სერვერზე

- `message` -- მომხმარებლის შეტყობინების გაგზავნა
- `cancel` -- აგენტის მიმდინარე ოპერაციის გაუქმება
- `ping` -- keepalive ping

### სერვერიდან კლიენტზე

- `token` -- სტრიმინგ პასუხის ტოკენი
- `tool_call` -- აგენტი ინსტრუმენტს იძახებს
- `tool_result` -- ინსტრუმენტის შესრულება დასრულდა
- `done` -- აგენტის პასუხი დასრულდა
- `error` -- შეცდომა მოხდა
- `pong` -- keepalive პასუხი

## კონფიგურაცია

```toml
[gateway.websocket]
max_connections = 100
ping_interval_secs = 30
max_message_size_kb = 1024
```
