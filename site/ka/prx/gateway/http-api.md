# HTTP API

PRX გეითვეი RESTful HTTP API-ს გამოაქვეყნებს აგენტის სესიების მართვის, შეტყობინებების გაგზავნისა და სისტემის სტატუსის მოთხოვნისთვის.

## საბაზო URL

ნაგულისხმევად, API ხელმისაწვდომია `http://127.0.0.1:3120/api/v1` მისამართზე.

## ენდფოინთები

### სესიები

| მეთოდი | ბილიკი | აღწერა |
|--------|--------|--------|
| `POST` | `/sessions` | ახალი აგენტის სესიის შექმნა |
| `GET` | `/sessions` | აქტიური სესიების ჩამოთვლა |
| `GET` | `/sessions/:id` | სესიის დეტალების მიღება |
| `DELETE` | `/sessions/:id` | სესიის შეწყვეტა |

### შეტყობინებები

| მეთოდი | ბილიკი | აღწერა |
|--------|--------|--------|
| `POST` | `/sessions/:id/messages` | შეტყობინების გაგზავნა აგენტისთვის |
| `GET` | `/sessions/:id/messages` | შეტყობინებების ისტორიის მიღება |

### სისტემა

| მეთოდი | ბილიკი | აღწერა |
|--------|--------|--------|
| `GET` | `/health` | ჯანმრთელობის შემოწმება |
| `GET` | `/info` | სისტემის ინფორმაცია |
| `GET` | `/metrics` | Prometheus მეტრიკები |

## ავთენტიფიკაცია

API მოთხოვნები bearer ტოკენს მოითხოვს:

```bash
curl -H "Authorization: Bearer <token>" http://localhost:3120/api/v1/sessions
```
