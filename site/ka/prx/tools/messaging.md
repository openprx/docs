# შეტყობინებები

PRX ორ შეტყობინებების ინსტრუმენტს გთავაზობთ. `message_send` ინსტრუმენტი მაღალი დონის ინტერფეისია ტექსტური, მედია და ხმოვანი შეტყობინებების გაგზავნისთვის ნებისმიერ კონფიგურირებულ არხზე, ხოლო `gateway` ინსტრუმენტი დაბალი დონის წვდომას უზრუნველყოფს Axum HTTP/WebSocket გეითვეიზე.

## კონფიგურაცია

შეტყობინებების ინსტრუმენტებს გამოყოფილი კონფიგურაციის სექცია არ აქვს. მათი ხელმისაწვდომობა არხისა და გეითვეის კონფიგურაციაზეა დამოკიდებული.

## ინსტრუმენტების მითითება

### message_send

შეტყობინების გაგზავნა ნებისმიერ კონფიგურირებულ არხსა და ადრესატზე. ინსტრუმენტი ავტომატურად მარშრუტიზებს აქტიურ არხზე.

```json
{
  "name": "message_send",
  "arguments": {
    "text": "აწყობა წარმატებით დასრულდა. ყველა 42 ტესტი ჩაბარდა.",
    "channel": "telegram"
  }
}
```

| პარამეტრი | ტიპი | სავალდებულო | ნაგულისხმევი | აღწერა |
|-----------|------|----------|---------|-------------|
| `text` | `string` | პირობითი | -- | ტექსტური შეტყობინების კონტენტი |
| `channel` | `string` | არა | აქტიური არხი | სამიზნე არხის სახელი |
| `media_path` | `string` | არა | -- | მედია ფაილის ბილიკი |
| `voice_path` | `string` | არა | -- | ხმოვანი/აუდიო ფაილის ბილიკი |

## უსაფრთხოება

გამავალი შეტყობინებები იგივე არხის პოლიტიკების ქვეშ მოქმედებს, რაც შემომავალი. აგენტს მხოლოდ კონფიგურირებულ და აქტიურ არხებზე შეუძლია გაგზავნა.
