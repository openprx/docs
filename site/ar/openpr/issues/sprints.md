# إدارة السبرينت

السبرينتات تكرارات محدودة الوقت لتنظيم العمل وتتبعه. ينتمي كل سبرينت إلى مشروع وله تاريخ بدء وتاريخ انتهاء ومجموعة مهام مُكلَّفة.

## إنشاء سبرينت

### عبر واجهة الويب

1. انتقل إلى مشروعك.
2. انتقل إلى قسم **Sprints**.
3. انقر **New Sprint**.
4. أدخل اسم السبرينت وتاريخ البدء وتاريخ الانتهاء.

### عبر API

```bash
curl -X POST http://localhost:8080/api/projects/<project_id>/sprints \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{
    "name": "Sprint 1",
    "start_date": "2026-03-24",
    "end_date": "2026-04-07"
  }'
```

### عبر MCP

```json
{
  "method": "tools/call",
  "params": {
    "name": "sprints.create",
    "arguments": {
      "project_id": "<project_uuid>",
      "name": "Sprint 1",
      "start_date": "2026-03-24",
      "end_date": "2026-04-07"
    }
  }
}
```

## حقول السبرينت

| الحقل | النوع | مطلوب | الوصف |
|-------|-------|-------|-------|
| الاسم | string | نعم | اسم السبرينت (مثل "Sprint 1" أو "Q1 Week 3") |
| تاريخ البدء | date | لا | تاريخ بدء السبرينت |
| تاريخ الانتهاء | date | لا | تاريخ انتهاء السبرينت |
| الحالة | enum | تلقائي | نشط، مكتمل، أو مخطط |

## تكليف المهام للسبرينت

كلِّف المهام لسبرينت بتحديث `sprint_id` للمهمة:

```bash
curl -X PATCH http://localhost:8080/api/issues/<issue_id> \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <token>" \
  -d '{"sprint_id": "<sprint_uuid>"}'
```

أو عبر واجهة الويب، اسحب المهام إلى قسم السبرينت أو استخدم لوحة تفاصيل المهمة.

## سير عمل تخطيط السبرينت

سير عمل تخطيط سبرينت نموذجي:

1. **إنشاء السبرينت** بتواريخ البدء والانتهاء.
2. **مراجعة المتراكمات** -- تحديد المهام للتضمين.
3. **نقل المهام** من Backlog/To Do إلى السبرينت.
4. **تحديد الأولويات** والمُكلَّفين لمهام السبرينت.
5. **بدء السبرينت** -- يبدأ الفريق العمل.
6. **تتبع التقدم** على اللوحة وعرض السبرينت.
7. **إكمال السبرينت** -- مراجعة العناصر المنجزة/المتبقية.

## أدوات MCP

| الأداة | المعاملات | الوصف |
|--------|----------|-------|
| `sprints.list` | `project_id` | سرد السبرينتات في مشروع |
| `sprints.create` | `project_id`, `name` | إنشاء سبرينت مع تواريخ اختيارية |
| `sprints.update` | `sprint_id` | تحديث الاسم أو التواريخ أو الحالة |
| `sprints.delete` | `sprint_id` | حذف سبرينت |
