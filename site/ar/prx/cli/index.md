# مرجع سطر الأوامر

ملف `prx` التنفيذي هو نقطة الدخول الوحيدة لجميع عمليات PRX -- المحادثة التفاعلية، إدارة الخادم، إدارة القنوات، وتشخيصات النظام.

## الرايات العامة

تُقبل هذه الرايات من جميع الأوامر الفرعية.

| الراية | اختصار | القيمة الافتراضية | الوصف |
|--------|--------|-------------------|-------|
| `--config` | `-c` | `~/.config/prx/config.toml` | مسار ملف الإعدادات |
| `--log-level` | `-l` | `info` | مستوى تفصيل السجل: `trace`، `debug`، `info`، `warn`، `error` |
| `--no-color` | | `false` | تعطيل المخرجات الملونة |
| `--quiet` | `-q` | `false` | إخفاء المخرجات غير الأساسية |
| `--help` | `-h` | | طباعة معلومات المساعدة |
| `--version` | `-V` | | طباعة الإصدار |

## الأوامر

| الأمر | الوصف |
|-------|-------|
| [`prx agent`](./agent) | تفاعل مع النموذج اللغوي لدورة واحدة (متوافق مع الأنابيب) |
| [`prx chat`](./chat) | محادثة طرفية غنية مع بث الاستجابات والسجل |
| [`prx daemon`](./daemon) | تشغيل بيئة PRX الكاملة (بوابة + قنوات + مهام دورية + تطور) |
| [`prx gateway`](./gateway) | خادم بوابة HTTP/WebSocket مستقل |
| [`prx onboard`](./onboard) | معالج الإعداد التفاعلي |
| [`prx channel`](./channel) | إدارة القنوات (عرض، إضافة، إزالة، تشغيل، فحص) |
| [`prx cron`](./cron) | إدارة المهام الدورية (عرض، إضافة، إزالة، إيقاف، استئناف) |
| [`prx evolution`](./evolution) | عمليات التطور الذاتي (الحالة، السجل، الإعدادات، التشغيل) |
| [`prx auth`](./auth) | إدارة ملفات مصادقة OAuth (تسجيل الدخول، التحديث، تسجيل الخروج) |
| [`prx config`](./config) | عمليات الإعدادات (المخطط، التقسيم، الدمج، القراءة، الكتابة) |
| [`prx doctor`](./doctor) | تشخيصات النظام (صحة الخادم، حالة القنوات، توفر النماذج) |
| [`prx service`](./service) | إدارة خدمة systemd/OpenRC (تثبيت، تشغيل، إيقاف، الحالة) |
| [`prx skills`](./skills) | إدارة المهارات (عرض، تثبيت، إزالة) |
| `prx status` | لوحة معلومات حالة النظام |
| `prx models refresh` | تحديث كتالوجات نماذج المزودين |
| `prx providers` | عرض جميع مزودي النماذج اللغوية المدعومين |
| `prx completions` | توليد الإكمال التلقائي للصدفة (bash، zsh، fish) |

## أمثلة سريعة

```bash
# الإعداد لأول مرة
prx onboard

# بدء محادثة تفاعلية
prx chat

# استعلام لدورة واحدة (قابل للبرمجة)
echo "Summarize this file" | prx agent -f report.pdf

# تشغيل الخادم مع جميع الخدمات
prx daemon

# فحص صحة النظام
prx doctor
```

## الإكمال التلقائي للصدفة

أنشئ ملفات الإكمال التلقائي لصدفتك وأضفها إلى ملف التهيئة:

```bash
# Bash
prx completions bash > ~/.local/share/bash-completion/completions/prx

# Zsh
prx completions zsh > ~/.zfunc/_prx

# Fish
prx completions fish > ~/.config/fish/completions/prx.fish
```

## متغيرات البيئة

يحترم PRX متغيرات البيئة التالية (تتجاوز قيم ملف الإعدادات):

| المتغير | الوصف |
|---------|-------|
| `PRX_CONFIG` | مسار ملف الإعدادات (مثل `--config`) |
| `PRX_LOG` | مستوى السجل (مثل `--log-level`) |
| `PRX_DATA_DIR` | مجلد البيانات (افتراضي: `~/.local/share/prx`) |
| `ANTHROPIC_API_KEY` | مفتاح API لمزود Anthropic |
| `OPENAI_API_KEY` | مفتاح API لمزود OpenAI |
| `GOOGLE_API_KEY` | مفتاح API لمزود Google Gemini |
