# prx onboard

شغّل معالج الإعداد لضبط PRX للاستخدام الأول. يرشدك المعالج عبر اختيار المزود وإعداد مفتاح API وضبط القنوات والتفضيلات الأساسية.

## الاستخدام

```bash
prx onboard [OPTIONS]
```

## الخيارات

| الراية | اختصار | القيمة الافتراضية | الوصف |
|--------|--------|-------------------|-------|
| `--quick` | `-q` | `false` | الوضع السريع -- أسئلة قليلة، إعدادات افتراضية معقولة |
| `--provider` | `-P` | | اختيار مسبق للمزود (تجاوز خطوة اختيار المزود) |
| `--config` | `-c` | `~/.config/prx/config.toml` | مسار ملف الإعدادات الناتج |
| `--force` | `-f` | `false` | الكتابة فوق ملف الإعدادات الحالي |
| `--non-interactive` | | `false` | الوضع غير التفاعلي (يتطلب `--provider` ومتغيرات البيئة للمفاتيح) |

## خطوات المعالج

يرشدك المعالج التفاعلي عبر الخطوات التالية:

1. **اختيار المزود** -- اختر مزود النماذج اللغوية الأساسي (Anthropic، OpenAI، Ollama، إلخ.)
2. **ضبط مفتاح API** -- أدخل وتحقق من مفتاح API
3. **اختيار النموذج** -- اختر نموذجًا افتراضيًا من المزود المختار
4. **إعداد القنوات** (اختياري) -- اضبط قناة مراسلة واحدة أو أكثر
5. **واجهة الذاكرة** -- اختر مكان تخزين ذاكرة المحادثة (markdown، SQLite، PostgreSQL)
6. **الأمان** -- إعداد رمز الاقتران وتفضيلات صندوق الرمل
7. **مراجعة الإعدادات** -- معاينة الإعدادات المولّدة والتأكيد

## أمثلة

```bash
# المعالج التفاعلي الكامل
prx onboard

# إعداد سريع مع Anthropic
prx onboard --quick --provider anthropic

# غير تفاعلي (مفتاح API من متغير البيئة)
export ANTHROPIC_API_KEY="sk-ant-..."
prx onboard --non-interactive --provider anthropic

# كتابة الإعدادات في مسار مخصص
prx onboard --config /etc/prx/config.toml

# إعادة تشغيل المعالج (الكتابة فوق الإعدادات الحالية)
prx onboard --force
```

## الوضع السريع

يتجاوز الوضع السريع (`--quick`) الخطوات الاختيارية ويستخدم إعدادات افتراضية معقولة:

- واجهة الذاكرة: SQLite
- الأمان: صندوق الرمل مفعّل، لا حاجة للاقتران
- القنوات: لا شيء (أضف لاحقًا باستخدام `prx channel add`)
- التطور: معطّل (فعّله لاحقًا في الإعدادات)

هذه أسرع طريقة للحصول على إعدادات عاملة:

```bash
prx onboard --quick --provider ollama
```

## ما بعد الإعداد

بعد اكتمال الإعداد الأولي، يمكنك:

```bash
# التحقق من الإعدادات
prx doctor

# بدء المحادثة
prx chat

# إضافة المزيد من القنوات
prx channel add

# تشغيل الخادم الكامل
prx daemon
```
