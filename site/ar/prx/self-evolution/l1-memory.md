# L1: تطور الذاكرة

الطبقة 1 هي طبقة التطور الذاتي الأكثر تكرارًا والأقل مخاطرة. تعمل على نظام ذاكرة الوكيل عبر ضغط الإدخالات المكررة تلقائيًا وتجميع الذكريات المرتبطة حسب الموضوع.

## نظرة عامة

يعمل تطور L1 بعد كل جلسة (أو وفق جدول قابل للضبط) وينفذ:

- **Compaction** -- دمج عدة إدخالات ذاكرة مرتبطة في ملخصات موجزة
- **Topic clustering** -- تجميع الذكريات حسب التشابه الدلالي
- **Relevance scoring** -- ضبط أوزان الذاكرة بناءً على تكرار الوصول
- **Pruning** -- إزالة الذكريات التي أصبحت قديمة أو متناقضة

## آلية العمل

1. بعد انتهاء الجلسة، يحلل L1 الذكريات المخزنة حديثًا
2. يحدد مجموعات الإدخالات المرتبطة باستخدام تشابه embeddings
3. تُضغط المجموعات التي تتجاوز حد الحجم إلى ملخصات
4. تُحدّث درجات صلة الذاكرة بناءً على تكرار الاستدعاء
