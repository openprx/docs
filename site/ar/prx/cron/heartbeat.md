# نبضة الحياة

نبضة الحياة هي فحص صحة دوري يراقب الحالة التشغيلية لخدمة PRX daemon. تعمل على فاصل زمني قابل للإعداد (الافتراضي: 30 ثانية) وتبلّغ عن صحة النظام.

## ما الذي يتم فحصه

- **عملية daemon** -- هل daemon يستجيب
- **اتصال المزوّدين** -- هل يمكن الوصول إلى مزوّدي LLM المُعدّين
- **استخدام الذاكرة** -- هل استهلاك الذاكرة ضمن الحدود
- **مساحة القرص** -- هل تتوفر مساحة قرص كافية لتخزين البيانات
- **الجلسات النشطة** -- عدد جلسات الوكيل العاملة وحالتها

## حالة الصحة

تنشر نبضة الحياة الحالة عبر:

- سجلات على مستوى debug
- نقطة نهاية API ‏`/health`
- مقاييس Prometheus (عند التمكين)
- عنوان URL خارجي اختياري لفحص الصحة

## الإعدادات

```toml
[cron.heartbeat]
interval_secs = 30
check_providers = true
check_disk_space = true
disk_space_threshold_mb = 100
external_health_url = ""  # optional: POST status to external URL
```
