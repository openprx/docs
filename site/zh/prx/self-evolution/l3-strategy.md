# L3: 策略进化

L3 是影响最大、频率最低的进化层。它修改 Agent 的战略行为 -- 工具访问策略、LLM 路由规则和治理参数。由于影响范围广，L3 变更始终需要显式批准。

## 概述

L3 进化处理：

- **工具策略调优** -- 调整可用工具及其权限级别
- **路由优化** -- 根据性能数据精炼模型选择启发式
- **治理参数** -- 调优安全阈值、速率限制和审批工作流
- **成本优化** -- 在质量和资源使用之间取得平衡

## 工作原理

1. L3 在数周/数月内收集聚合性能数据
2. 识别模式（例如，更便宜的模型可以充分处理 80% 的查询）
3. 提出带有预期影响分析的策略变更
4. 变更排队等待人工批准
5. 批准的变更带有自动回滚能力地应用

## 配置

```toml
[self_evolution.l3]
enabled = false
schedule = "monthly"
require_approval = true
rollback_window_hours = 168  # 7 天
max_policy_changes_per_cycle = 3
```
